1. WRITE A PRORAM TO FIND THE LETTER WHICH REPEATED MOST NUMBER OF TIMES IN A GIVEN STRING.
2. PRINT THE FOLLOWING PATTERN FOR A GIVEN INTEGER ‘n’.
  EXAMPLE:  FOR n=4,       * * * *
                           *     *
                           *     *
                           * * * * 
3.PRINT ‘n’ ROWS IN PASCAL’S TRIANGLE.
4.CHECK IF A GIVEN SUB-STRING IS PRESENT IN A GIVEN STRING.
5.PRINT REVERSE OF A STRING INSIDE THE PARANTHESES.
  EXAMPLE: i)GIVEN STRING=’HELLO(REV)WORLD’ SHOULD RETURN ‘HELLOVERWORLD’.
  ii)‘HELLO((REV))WORLD’ SHOULD RETURN ‘HELLOREVWORLD’ AS ‘REV’ IS REVERSED TWICE.
  iii)’HELLO((REV)ERSE)WORLD’ GIVES ’HELLOESREREVWORLD’  (’HELLO(VERERSE)WORLD’  ->  ’HELLOESREREVWORLD’).
6.GIVEN AN ARRAY OF INTEGERS,FIND THE MAXIMUM PRODUCT OF TWO NUMBERS.
  EXAMPLE: ARRAY a={2,5,7,3,-10,-8} SHOULD RETURN 80.
7.PROBLEM 6 WITH ONLY POSITIVE INTEGERS.
8.GIVEN A SENTENCE,FIND THE LONGEST WORD IN IT.
9.GIVEN A NUMBER,FIND A PRIME NUMBER NEAREST TO IT.
10.GIVEN TWO ARRAYS NAMELY PRICE AND QUANTITY,FIND THE TOTAL COST.
  EXAMLE:GIVEN PRICE={5,10,15},QUANTITY={2,3,0} RETURN 40 AS COST=(5*2+10*3+15*0).
